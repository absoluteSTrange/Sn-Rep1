HTML Headers (h1 , h2, etc)

h1 being the most largest and the h6 being the least largest

The code sample below shows all the headings in html 

        
          <!DOCTYPE html>
          <html lang="en">
          <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Learning About Headers</title>
          </head>
          <body>

            <h5>Hey! I am a h1</h6>
            <h6>Hey! I am a h1</h6>
          </body>
          </html>
